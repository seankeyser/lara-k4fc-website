## Projects

### 🏆 LARA K4FC Club Website
**Live:** [seankeyser.github.io/lara-k4fc-website](https://seankeyser.github.io/lara-k4fc-website)  
**Repo:** [github.com/seankeyser/lara-k4fc-website](https://github.com/seankeyser/lara-k4fc-website)

A full redesign of the Lake Amateur Radio Association website — a real club with real members in Lake County, Florida. Built from scratch as a single-page HTML/CSS/JS application with a focus on visual quality, mobile responsiveness, and live data integration.

**Key features:**
- Live HF band conditions widget pulling real-time solar data from hamqsl.com (SFI, SN, A-Index, K-Index)
- Fully responsive mobile layout with animated hamburger navigation
- Club history timeline section
- Repeater network reference table (FM Analog + DMR)
- ARES/LCARES emergency communications section
- Membership application and dues payment modal forms
- Scroll-triggered animations and active nav state tracking

**Tech used:** HTML5 · CSS3 (custom properties, grid, flexbox) · Vanilla JavaScript · Fetch API · DOMParser · GitHub Pages

### 🏥 Patient Transfer Command Center *(In Development)*

A healthcare dashboard application built on my direct experience coordinating inpatient transfers at the University of Pennsylvania Hospital. Simulates the real-time workflow of a transport communications coordinator — managing transfer requests, tracking transport status across air and ground, and logging activity history.

**Planned features:**
- Live transfer queue with status tracking (Requested → Dispatched → In Transit → Arrived)
- Priority classification (Critical / Urgent / Routine)
- Transport mode filtering (Air / Ground)
- Transfer activity log with timestamps
- New transfer intake form
- Stats bar (daily volume, response time calculations)
- React-based component architecture with localStorage persistence

**Why I'm building this:** Every field in this UI represents a decision I made in a real communications center. This project is about encoding that operational knowledge into software that could genuinely improve how transport teams work.

**Tech planned:** React · Vite · CSS Modules · localStorage → MySQL backend
